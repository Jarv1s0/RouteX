# 项目核心知识库
## 项目目标
维护 RouteX (基于 Sparkle/Mihomo 的 Electron Clash 客户端)。
主打：双栏设置布局、响应式设计、各色高级统计与工具。

## 核心共识
- **项目名称**: RouteX
- **版本**: 1.7.5
- **技术栈**:
  - **Core**: Electron, React 19, TypeScript, Vite
  - **UI**: HeroUI (v2.8.5), TailwindCSS v4, Framer Motion
  - **Editor**: Monaco Editor
  - **Package Manager**: pnpm 10.x
- **核心依赖**:
  - `mihomo` (backend likely, embedded)
  - `sparkle` (upstream)
- **主要特性**:
  - 双栏设置
  - 规则颜色编码
  - 丰富的统计图表
