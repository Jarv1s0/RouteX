## ✨ 主要特性

🎨 全新的双栏设置页面布局  
📱 响应式设计，更好的用户体验  
📂 重新组织的设置层次结构  
🎯 主开关+二级菜单的直观操作模式  
💫 优化的视觉样式和交互效果  
🔍 改进的输入框可见性  
🏷️ 规则页面颜色编码  
📊 强大的统计功能（流量、订阅、网络健康度、规则效率）  
🛠️ 实用的工具集（DNS、规则测试、连通性检测、流媒体解锁检测）  
🔗 增强的连接管理（隐藏、暂停、分组、表格视图）  
🎨 精美的 UI 设计和颜色标识系统
