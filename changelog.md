# 更新日志

## v2.2.4

### ✨ 新功能
- 新增覆写回滚入口，出现异常时可以更快恢复并重新拉起内核。
- 工具页与订阅页加入远程图标本地缓存，减少重复下载与加载闪烁。
- 工具页“全部测试”加入并发上限控制，避免一次性打满检测请求。
- 自动构建版本统一追加 `-AutoBuild` 标识，方便识别构建来源。

### ⚡ 优化
- 主应用监听改为按页面启停，减少后台无效事件处理。
- SSID 检测改为按配置懒启动，并支持配置变更后的动态刷新。
- 网络健康监控收敛到主进程，补充 DNS 延迟采样，减少渲染层重复探测。
- 连接页过滤与排序链路增加缓存，高连接数场景下更流畅。
- 应用更新检查改为启动检查 + 回前台按时间窗补刷，降低固定轮询消耗。
- 订阅与覆写配置更新链路进一步收敛，减少重复写入和同步逻辑。
- 主进程入口职责拆分，启动、退出、深链和窗口生命周期更清晰。

### 🛠 修复
- 修复 IPC 错误包装器在返回 `null` 时触发告警的问题。
- 修复统计页规则详情弹窗导致无限重渲染的问题。
- 修复 Windows 自定义无边框窗口下右上角按钮点击冲突的问题。
- 清理流量统计与网络 IPC 的类型和副作用问题，恢复 `typecheck` 与 `build` 验证链路。
- 升级 `yaml` 并锁定 `brace-expansion`、`picomatch` 安全版本，清理 Dependabot 依赖告警。

## v2.2.3

### ✨ 新功能
- 统计页新增按账期展示的订阅数据卡，支持年份账期选择和关键日期标记。

### ⚡ 优化

- 调整主窗口、侧栏卡片和统计图表的视觉表现，整体交互更稳定统一。
- 优化代理组测速默认行为，减少进入页面时的无效测速。
- 统一全站图标到 Lucide，界面细节更一致。
- 规则首页统计口径调整为“规则集条目 + 手写规则”，数据更准确。

### 🛠 修复
- 修复开发模式白屏问题，并补齐沙箱下的 preload 桥接。
- 修复标题栏覆盖层报错和嵌套 `button` 告警。
- 优化实时流量图悬浮提示，减少提示瞬间消失的问题。

### 🧹 调整
- 删除容易误导的“地理分布”页面，仅保留网络拓扑视图。
- README 中的许可证说明已与仓库实际 `LICENSE` 对齐。

## 🗂 历史更新

### ⚡ 性能优化
- 精简主窗口恢复逻辑，减少白屏探测带来的额外负担。
- 优化 SubStore 首页缓存、连接页资源预加载和地理定位缓存策略。
- 清理拓扑页与流量监控中的无效拉取和重复启动逻辑。
- 收敛依赖与构建链路，降低构建和运行时开销。

### 🛠 稳定性修复
- 托盘、悬浮窗与网络监听统一改为更安全的注册与解绑方式。
- 应用退出前统一等待关键数据落盘和系统代理回滚完成。
- `providerStats` 改为串行队列执行，避免定时任务与手动操作互相覆盖。
- 统一前端错误提示入口，减少重复处理与状态残留。

### 🎨 UI 与交互
- 统一弹窗样式、关闭按钮和二级弹窗头部交互。
- 清理散落的 `alert()`，统一接入全局提示与对话框。
- 收窄全局 focus 样式影响范围，避免误伤日志复制等操作。

### 🔐 安全修复
- 升级 `tar`、`flatted`、`yauzl` 等依赖，清理现有高危与中危告警。
