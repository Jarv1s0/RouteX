# 更新日志

## v4.0.0

RouteX `4.0.0` 是桌面端重写后的第一个正式版本。本版本迁移到 Tauri 2，重新接入 Mihomo 运行时、配置管理、规则维护、连接查看、流量统计、网络工具、系统托盘、全局快捷键、深链路和应用内更新。

### 版本定位

- 这是 RouteX 桌面端重写后的首个正式版本，不是旧桌面版本的维护更新。
- 运行时迁移到 Tauri 2，窗口、资源、权限、托盘、快捷键、深链路、安装包和更新流程都已重新实现。
- 发布目标包括 Windows、macOS 和 Linux，实际安装包格式以本次 Release assets 为准。
- 本版本优先保证核心代理管理流程可用：安装、启动、运行 Mihomo、维护配置、切换策略、排查规则和查看连接。

### 新增与重构

- Mihomo 管理：支持内置核心、核心启动/停止/重启、控制器连接、运行日志、配置热重载、系统代理、TUN、DNS、Sniffer、环境变量和权限模式。
- 配置管理：支持本地配置、远程订阅、配置切换、排序、编辑、更新、状态展示、覆写配置绑定、JavaScript 覆写执行和 Proxy Provider 管理。
- 规则与策略：支持规则、规则 Provider、快速规则、从连接创建规则、代理组、节点筛选、节点测速、链路查看和代理组状态查看。
- 连接观测：支持活动连接、已关闭连接、连接详情、连接关闭、连接列表实时刷新和网络拓扑视图。
- 统计中心：重做统计页和应用面板，支持实时流量、历史流量、账期统计、规则命中、Provider 使用情况和流量排行。
- 网络工具：支持 IP 信息、连通性检测、规则匹配、DNS 查询和流媒体检测。
- 桌面能力：支持系统托盘、托盘菜单、浮动窗口、窗口置顶、退出确认、深链路协议、全局快捷键和跨平台安装包。
- 国际化：新增英文界面，保留简体中文作为默认使用场景。

### 体验优化

- 重做侧边栏、管理卡片、统计面板、流量排行和多个应用表面，提升信息密度和状态识别。
- 优化节点测速逻辑，自动测速会解析到叶子节点，避免嵌套策略组导致结果不准。
- 调整节点筛选、测速结果刷新、下拉菜单文案、连接页 Tab 切换和 CodeMirror 编辑器选区显示。
- 用 CodeMirror 替换 Monaco，减少编辑器资源体积，并改善 diff 滚动和折叠标记。
- 拆分前端启动包，延迟加载诊断和编辑器相关模块，降低首屏加载压力。

### 打包与稳定性

- 修正安装版资源路径，安装后可以读取内置核心、服务程序、geo 数据、托盘图标和安装器资源。
- 修正 Windows 安装/升级流程，安装器会等待 RouteX 进程退出，并在升级前停止 bundled Mihomo。
- 区分开发版、自动构建版和正式发布版的 Windows 任务计划名称，避免不同构建互相覆盖。
- 修正全局快捷键加载、提权启动、启动可见性、关闭行为、任务计划参数、应用数据目录和托盘图标加载。
- 修正系统代理、TUN、服务模式、权限弹窗、启动警告和卸载清理流程。
- 接入 Tauri updater，发布包会生成签名更新文件，并通过 `latest.json` 支持后续应用内更新。
- 稳定 Windows、macOS 和 Linux 构建流程，优化自动构建产物命名、Release 上传和 AppImage 构建。
- 增加本地验证、Rust 测试、Clippy、死代码检查、依赖审计和发布环境检查。

### 安装与升级说明

- Windows 下使用 TUN、系统代理、服务模式和任务计划相关功能时需要管理员权限，也可能受系统安全策略影响。
- Windows 用户升级或重新安装后，如遇到提权、TUN 或系统代理异常，请先删除任务计划程序中的旧 RouteX 任务，再以管理员身份启动 RouteX 重新注册。
- 可能需要清理的旧任务名包括 `routex`、`routex-run`、`routex-dev`、`routex-run-dev`、`routex-autobuild` 和 `routex-run-autobuild`。
- 清理旧任务或首次安装后，请右键 RouteX 选择“以管理员身份运行”启动一次；提权任务注册完成后，后续可正常双击启动。
- macOS 和 Linux 构建已经接入发布流程，但不同发行版、签名策略和系统权限可能仍有差异，请以本次 Release assets 和系统提示为准。

### 已知限制

- 首个正式版本以桌面端核心流程为优先，旧版本的个别边缘行为可能没有完全保留。
- 不支持从旧版本直接应用内升级；请下载本次 Release 安装包后重新安装。
- TUN、系统代理和服务模式属于高权限功能，异常时优先检查管理员权限、系统代理设置、安全软件拦截和旧任务计划残留。
