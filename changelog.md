# RouteX 更新日志

- 📊 订阅统计功能增强

## v1.6.0 (2025-12-22)

### 新功能
- 📊 新增统计页面
  - 实时流量速度监控
  - 24小时/7天/30天流量统计图表
  - 订阅流量统计（基于 subscription-userinfo）
- 🛠️ 新增工具页面
  - DNS 查询
  - 规则测试
  - 连通性检测
  - 流媒体解锁检测（ChatGPT、Gemini、Netflix、YouTube、Spotify、TikTok）
  - IP 信息查询

### 优化
- ✨ 连通性检测和流媒体检测添加动画效果
- 🖼️ 图标加载失败时显示默认图标
- 🔄 同步上游代码

## 下载说明

- **Windows**: 下载 `RouteX-*-windows-*-setup.exe` (安装版) 或 `RouteX-*-windows-*-portable.7z` (便携版)
- **Linux**: 下载对应发行版的包文件 (`.deb` 或 `.rpm`)
- **macOS**: 下载 `RouteX-*-macos-*.pkg` 文件

## 系统要求

- Windows 10/11 (x64)
- Linux (x64, arm64)
- macOS 10.15+ (x64, arm64)
