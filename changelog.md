# Changelog

## V1.8.1

### 🎨 UI & 交互升级
- **视觉焕新**: 全面采用 Fusion Design 玻璃拟态设计，增强代理组与节点卡片的视觉层次与通透感
- **细节优化**: 优化连接详情页可视度，统一全局组件样式，提升交互体验

### 📊 功能增强
- **Sub-Store 集成**: 新增侧边栏入口，支持自定义显示及主题自动跟随
- **日志系统**: 新增暂停/恢复控制，优化滚动查看体验，状态查看更便捷

### ⚡️ 性能与稳定性
- **核心重构**: 重构连接管理模块，彻底修复内存溢出崩溃问题，大幅提升稳定性
- **问题修复**: 修复统计页面白屏、Sub-Store 弹窗布局异常及 DNS 延迟显示错误
