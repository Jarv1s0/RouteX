# 更新日志

## v4.0.0

RouteX 桌面端已在 `4.0.0` 全量迁移到 Tauri，本版本同时完成桌面构建链路切换与迁移后的兼容性修复。

### 主要变更

- 桌面运行时从 Electron 迁移到 Tauri，并同步调整自动构建与手动发布流程。
- 补齐 Tauri 桌面能力接入，包括系统托盘、浮动窗口、托盘菜单、深链路与资源打包。
- 渲染层切换到新的桌面桥接实现，并恢复 Tauri 环境下的流量桥接降级路径。
- 推进迁移后的桌面交互流程，更新部分设置页、侧边栏卡片与默认配置。

### 修复

- 恢复需要提权场景下的重启流程，避免管理员权限操作中断。
- 修复托盘图标从资源目录读取的问题，降低打包后图标缺失风险。
- 修复 macOS 应用菜单编译失败问题。
- 修复 macOS 与 Linux 下的编译错误，并适配部分 Tauri 下暂不支持的界面能力。
- 对齐 NSIS 安装器资源，并在构建前自动创建 Tauri 资源目录。
