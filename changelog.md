# RouteX 更新日志

## v1.6.5

### 新功能
- 📊 新增统计页面
  - 实时流量速度监控
  - 24小时/7天/30天流量统计图表
  - 网络健康度监控（延迟、抖动、丢包率）
  - 规则效率统计
- 📊 订阅统计功能
  - 支持按订阅分别查看流量统计
  - 支持按月份查看历史数据
  - 柱状图显示每日增量流量
  - 鼠标悬停显示订阅名称和流量详情
  - 自动从 Profile 配置获取订阅信息
- 🛠️ 新增工具页面
  - DNS 查询
  - 规则测试
  - 连通性检测
  - 流媒体解锁检测（ChatGPT、Gemini、Netflix、YouTube、Spotify、TikTok）
  - IP 信息查询
- 🔍 连接页面功能增强
  - 支持手动隐藏连接（持久化保存）
  - 支持暂停/继续刷新连接列表
  - 隐藏连接规则管理（显示/清除）
  - 暂停状态视觉提示

### 优化
- 🎨 连接表格视图优化
  - 支持自定义字体大小
  - 支持列宽调整
  - 支持按列排序
  - 自动缩放以适应窗口大小
- 🎨 连接卡片样式优化
  - 更现代的卡片设计（阴影、边框、悬停效果）
  - 简洁的颜色标识系统
    - 已关闭连接: 红色
    - Inner 类型: 橙色
    - 其他活动连接: 灰色
    - 节点: 柔和的绿色
  - 进程名和目标地址在同一行显示，用箭头分隔
  - 进程名加粗，目标地址更易读
  - 流量数字颜色与统计页面一致（上传青色、下载紫色）
- 🎨 连接设置弹窗优化
  - 更大的弹窗尺寸
  - 更美观的布局和间距
  - 关闭按钮移至右上角
  - 统一的内边距
- 🎨 工具栏按钮优化
  - 关闭全部按钮改为简洁的图标按钮
  - 隐藏相关按钮统一为图标样式
- 🎨 侧边栏流量卡片颜色优化
  - 上传速度：青色
  - 下载速度：紫色
- 🐛 修复订阅名称修改后下拉菜单显示旧名称的问题
- ✨ 连通性检测和流媒体检测添加动画效果
- 🖼️ 图标加载失败时显示默认图标
- 🔄 同步上游代码

## 下载说明

- **Windows**: 下载 `RouteX-*-windows-*-setup.exe` (安装版) 或 `RouteX-*-windows-*-portable.7z` (便携版)
- **Linux**: 下载对应发行版的包文件 (`.deb` 或 `.rpm`)
- **macOS**: 下载 `RouteX-*-macos-*.pkg` 文件

## 系统要求

- Windows 10/11 (x64)
- Linux (x64, arm64)
- macOS 10.15+ (x64, arm64)
