## 更新日志（Happy Lunar New Year）

### 🛠️ 内核适配
- **适配 Mihomo v1.19.20**: 移除 Relay 支持，新增 Wildcard 规则与 DNS Rule 模式。

### ✨ 新增功能
- **快捷规则管理**: 连接列表右键快速创建路由规则。
- **自定义规则覆写**: 支持启用/禁用及 YAML/JS 格式。
- **网络拓扑**: 新增可视化连接链路展示。
- **MRS 预览**: 支持 Mihomo Rule Set 预览。

### 🚀 优化与修复
- **界面重构**: 全新流量圆环、状态卡片与统计排行榜样式。
- **交互提升**: 优化错误弹窗、WebDAV 操作及外部控制器体验。
- **问题修复**: 修复代理链内存泄漏、WebDAV 备份及 Axios 安全漏洞。
