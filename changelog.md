# Changelog

## V1.8.3

### ✨ 新增功能 (Features)
- **任务栏状态同步**: 任务栏图标现支持跟随系统代理/Tun 模式状态变化。

### ⚡️ 性能优化 (Performance)
- **统计页面重构**: 彻底修复了在高并发连接（如直播场景 2000+ 连接）下，统计页面卡死及内存泄漏的问题。
- **动画优化**: 优化了统计详情弹窗的过渡动画。
- **代理列表**: 移除重型动画并应用 Memoization，提升滚动帧率。
- **搜索体验**: 引入 `useDeferredValue` 优化搜索交互，消除阻塞感。
- **状态管理**: 核心模块迁移至 Zustand，降低空闲资源占用。
- **无阻塞启动**: 重构启动逻辑，移除同步 IO，修复白屏问题。
- **侧边栏重构**: 重写拖拽逻辑，解决掉帧与重渲染问题。
- **日志引擎**: 引入节流机制，支持批量更新。
- **资源调度**: 优化 IPC 通信，降低后台 CPU 占用。
- **路由懒加载**: 应用代码分割，减少首屏体积。

### 💄 界面优化 (UI/UX)
- **卡片交互优化**: 侧边栏卡片仅在内容区域显示手型指针，空白区域保持默认指针，提升交互直觉。
- **布局调整**: 移除了工具页面的 "IP 纯净度检测" 卡片。
- **文案修正**: 优化统计页面标题。
- **视觉反馈**: 新增连接页骨架屏，优化加载过渡。
- **容错机制**: 引入全局错误边界，提升崩溃恢复体验。

### 🛠 代码重构 (Refactoring)
- **架构瘦身**: 解耦 `App.tsx`，提取独立组件。
- **状态重构**: 全局配置管理迁移至 Zustand。
