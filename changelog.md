# V2.1.5 更新日志

## 🛠️ 内核适配 (Kernel Adaptation)

*   **适配 Mihomo v1.19.20**: 移除 Relay 支持，新增 Wildcard 规则与 DNS Rule 模式支持。

## 🚀 性能与体验优化 (Performance & UX)

*   **内核更新提示**: 左侧内核卡片支持显示新版本提示。
*   **新增网络拓扑**: 可视化展示连接链路，支持主题适配。
*   **新增 MRS 预览**: 支持 Mihomo Rule Set (MRS) 格式规则集的预览。
*   **优化外部控制器**: 修复 Zashboard/MetaCubeXD 面板下载与切换问题。
*   **优化发布日志**: 简化 Release 标题显示。
*   **优化错误弹窗**: 全新的美观模态框，支持置顶显示。
*   **优化连接详情**: 调整流量统计布局，新增"断开连接"功能。
*   **修复代理链问题**: 修复了代理链的内存泄漏、保存权限错误及环路检测逻辑。
*   **其他修复**: 修复 WebDAV 备份问题及安全漏洞。
