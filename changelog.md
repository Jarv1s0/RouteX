# RouteX 更新日志

## v1.6.6

### 优化
- 🎨 规则页面样式优化
  - 规则列表和规则集合卡片添加边框、阴影和悬停效果
  - 统一序号格式和间距
  - 规则集合显示顺序调整
- 🎨 日志页面样式优化
  - 日志卡片添加边框、阴影和悬停效果
  - 类型标签改用 Chip 组件，颜色更明显
  - 支持右键菜单复制完整日志
- 🎨 连接详情弹窗优化
  - 左侧 JSON 数据支持选择复制
- 🎨 弹窗关闭按钮统一优化
  - 所有设置类弹窗关闭按钮移至右上角
  - 使用统一的图标按钮样式

## 下载说明

- **Windows**: 下载 `RouteX-*-windows-*-setup.exe` (安装版) 或 `RouteX-*-windows-*-portable.7z` (便携版)
- **Linux**: 下载对应发行版的包文件 (`.deb` 或 `.rpm`)
- **macOS**: 下载 `RouteX-*-macos-*.pkg` 文件

## 系统要求

- Windows 10/11 (x64)
- Linux (x64, arm64)
- macOS 10.15+ (x64, arm64)
