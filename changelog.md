# Changelog

## V1.8.2

### ⚡️ 性能优化 (Performance)
- **侧边栏重构**: 采用直接 DOM 操作重写侧边栏拖拽逻辑，彻底解决拖拽时的掉帧与重渲染问题，操作丝般顺滑。
- **状态管理**: 引入 Zustand 重构全局配置管理，移除 Context Provider，显著降低应用重渲染频率。
- **路由懒加载**: 全面应用 Code Splitting 技术，显著减少首屏资源体积，提升应用启动速度与内存效率。
- **日志引擎**: 引入智能节流 (Throttling) 机制，支持 500ms 批量更新，彻底告别高频日志刷屏卡顿。
- **资源调度**: 优化主进程与渲染进程通信 (IPC)，大幅降低后台静默运行时的 CPU 占用。

### 🌈 用户体验 (User Experience)
- **视觉反馈**: 新增连接页面骨架屏 (Skeleton)，页面加载过渡更自然。
- **容错机制**: 引入全局错误边界 (Error Boundary)，通过“崩溃恢复”代替“白屏等待”，提升系统韧性。

### 🛠 代码重构 (Refactoring)
- **架构瘦身**: 深度解耦 `App.tsx`，将全局弹窗与侧边栏提取为独立组件 (`AppSidebar`, `GlobalConfirmModals`)，提升代码可维护性。
