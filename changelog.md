## 更新日志（Happy Lunar New Year）

### 🛠️ 内核适配

- **适配 Mihomo v1.19.20**: 移除 Relay 支持，新增 Wildcard 规则与 DNS Rule 模式支持。

### 🚀 性能与体验优化

- **内核更新提示**: 左侧内核卡片支持显示新版本提示。
- **新增网络拓扑**: 可视化展示连接链路，支持主题适配。
- **新增 MRS 预览**: 支持 Mihomo Rule Set (MRS) 格式规则集的预览。
- **优化外部控制器**: 修复 Zashboard/MetaCubeXD 面板下载与切换问题。
- **优化发布日志**: 简化 Release 标题显示。
- **优化错误弹窗**: 全新的美观模态框，支持置顶显示。
- **优化连接详情**: 调整流量统计布局，新增"断开连接"功能。
- **优化统计排行榜**: 全新排行榜样式，支持策略/进程/主机切换，优化对齐与显示。
- **优化拓扑图渲染**: 使用 SVG 渲染器提升网络拓扑文字清晰度；地理分布图提升 DPI 适配。
- **优化覆写样式**: 统一覆写卡片启用状态下的图标与时间颜色。
- **修复代理链问题**: 修复了代理链的内存泄漏、保存权限错误及环路检测逻辑；修复因前置节点缺失导致的配置检查失败。
- **其他修复**: 修复 WebDAV 备份问题及安全漏洞。
- **优化右键菜单**: 解决列表滚动菜单错位问题；优化新建规则按钮视觉样式。

### ✨ 新增功能

- **快捷规则管理**: 支持在连接列表中右键快速创建路由规则，自动填充域名/IP/进程信息。
- **自定义规则覆写**: 优化覆写交互，支持菜单控制启用/禁用，新增绿底状态标识；支持 YAML/JS 格式。
- **性能优化**: 连接页面支持暂停刷新，降低高频数据更新时的性能消耗。

### 🎨 界面与交互优化

- **流量圆环重构 (Pro Max)**: 升级为双层实心圆环（100% 填充），支持内环（本次）与外环（今日）对比显示。移除高斯模糊光效，采用纯色与半透明结合的现代设计；字体与圆环尺寸实现全响应式缩放。
- **状态卡片升级**: 状态网格（运行时、连接数、内存）采用语义化配色（青/紫/绿）与等宽字体，提升数据可读性与仪表盘质感。
- **布局对称性优化**: 流量统计模块采用左（上传）-中（圆环）-右（下载）的三栏对称布局，视觉更加平衡。

- **输入框样式统一**: 全局配置、端口设置、外部控制器等页面采用统一的圆角与阴影风格。
- **界面去冗余**: 移除外部控制器、环境变量及外观设置页面的灰色背景容器，实现扁平化布局。
- **WebDAV 弹窗优化**: 底部操作按钮样式升级，统一交互体验。
- **外部控制器布局**: 优化配置项分割线与输入框位置，提升可读性。
