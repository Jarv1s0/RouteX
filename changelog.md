# RouteX 更新日志

## v1.6.18

### 性能优化
- 优化流量图性能表现，实现窗口失焦自动暂停
- 优化连接页面表格渲染，使用虚拟滚动提升大量连接时的性能
- 优化 Stats 页面窗口失焦时暂停轮询
- 优化连接卡片时间更新，使用统一定时器替代独立定时器

### UI 优化
- 统一规则列表和规则集合卡片样式
- 优化外部控制器页面布局，简化嵌套层级
- 修复 SettingCard 无标题时上边距问题
- 连接卡片添加 hover 效果
- Tun 类型连接使用蓝色标识
