# 更新日志

## v4.3.0

### 新增

- 适配 mihomo `v1.19.29`，支持更多代理配置。
- 自动构建版本会显示版本标记、构建编号和提交信息。
- 正式版和自动构建版本可分别检查更新，并支持手动切换。

### 修复

- 修复订阅自动更新没有按计划执行的问题。
- 更新 ZIP 处理组件，修复可能占用过多内存的问题。
- 更新部分依赖，修复已知安全问题。
- 修复悬浮窗部分功能无法正常使用的问题。
- 修复悬浮窗网速和图标动画跳动的问题。

### 优化

- 简化应用数据目录，升级后会自动迁移旧数据。
- “打开配置文件目录”现在会直接打开配置文件夹。
- 减少配置检查时产生的重复文件和缓存占用。
- 多个订阅同时更新时减少重复提示和内核重启。
- 移除任务栏、状态栏网速显示和 TrafficMonitor 集成。
- 减少开发模式下的重复文件监听。
- 调整悬浮窗外观，并统一更新按钮图标。
