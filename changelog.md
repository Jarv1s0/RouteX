# V2.1.3 更新日志

## 🚀 性能与稳定性修复 (Performance & Stability Fixes)

本次更新主要修复了由于代理链环路导致的严重内存泄漏问题，并解决了生产环境下的图标同步异常。

### 🐛 深度修复 (Critical Fixes)
*   **代理链环路检测增强 (Critical Memory Leak Fix)**:
    *   修复了代理链配置中可能存在的隐蔽环路问题，该问题曾导致系统消耗 20GB+ 内存并卡死。
    *   现在 `injectChainProxies` 能正确识别并阻断包含普通代理（通过 `dialer-proxy` 引用）的循环依赖。
*   **其他修复**:
    *   **内存泄漏治愈**: 修复规则页面 (`Rules`) 及统计页面 (`Stats`) 的内存占用问题。
    *   **图标同步修复**: 重构托盘与任务栏图标加载逻辑，采用静态资源导入，解决生产环境显示问题。
