# Changelog

## V1.8.2

### ✨ 新增功能 (Features)
- **IP 纯净度检测**: 新增 IP 风险检测卡片，支持自动路由、手动节点选择及多源切换。
- **图标跟随**: 任务栏图标现已支持与托盘图标实时同步状态颜色。

### ⚡️ 性能优化 (Performance)
- **代理列表**: 优化渲染逻辑，显著提升长列表滚动流畅度。
- **搜索体验**: 优化搜索交互，消除快速输入时的 UI 阻塞感。
- **状态管理**: 核心模块迁移至 Zustand，降低资源占用。
- **无阻塞启动**: 重构启动逻辑，修复启动白屏问题。
- **侧边栏重构**: 优化拖拽逻辑，解决掉帧与重渲染问题。
- **日志引擎**: 引入节流机制，优化日志更新性能。
- **资源调度**: 优化 IPC 通信，降低后台静默时的 CPU 占用。
- **路由懒加载**: 应用代码分割，减少首屏资源体积。

### 🌈 用户体验 (User Experience) / 💄 界面优化 (UI/UX)
- **布局调整**: 优化“工具”页面卡片排列，IP 检测优先展示。
- **视觉反馈**: 新增连接页骨架屏，加载过渡更自然。
- **容错机制**: 引入全局错误边界，提升应用稳定性。

### 🛠 代码重构 (Refactoring)
- **架构瘦身**: 解耦 `App.tsx`，提取独立组件。
- **状态重构**: 全局配置管理迁移至 Zustand。
