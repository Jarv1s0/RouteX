# V2.1.4 更新日志

## 🚀 性能与体验优化 (Performance & UX)

*   **修复内存泄漏**: 解决了由代理链环路配置导致的高内存占用问题。
*   **修复图标同步**: 彻底解决了 Windows 11 下任务栏与托盘图标不一致的问题。
*   **优化错误弹窗**: 使用全新的美观模态框替换了系统原生错误提示，支持置顶显示。 
*   **优化连接详情页**: 重新设计了连接详情弹窗布局，移除冗余分割线，调整流量统计显示位置，并新增了“断开连接”功能。 
*   **修复代理链配置**: 修复了无法使用订阅节点（Provider Node）作为代理链前置节点的问题，并增强了针对策略组的环路检测逻辑。
*   **修复 WebDAV 备份**: 修复了 WebDAV 备份功能未包含代理链配置 (`chains.yaml`) 的问题。
*   **修复代理链保存**: 修复了编辑代理链保存时因管道未释放导致的“权限拒绝”错误，增加了内核重启重试机制。
*   **优化代理链逻辑**: 实施了全局代理链隔离策略，确保代理链不会被自动筛选规则误吸入，彻底解决了因正则匹配导致的死循环崩溃问题。
*   **新增 MRS 预览支持**: 实现了对 Mihomo Rule Set (MRS) 格式规则集的预览功能，支持自动调用内核进行解码显示。
*   **安全更新**: 修复了 `@isaacs/brace-expansion` 的安全漏洞 (GHSA-7h2j-956f-4vf2)。
