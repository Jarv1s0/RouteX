# 更新日志

## v4.2.0

> 重要升级提示：如果你当前使用的是 `v4.1.x`，升级前请先备份数据，卸载旧版本后再安装 `v4.2.0`。安装完成后需要重新拉取订阅；直接覆盖安装可能导致旧配置无法承接。

### 新增

- 更新 Mihomo 配置模板。
- 增加发布前检查，减少内部配置不一致带来的问题。

### 修复

- 修复部分依赖的安全风险。
- 修复拓扑图部分连接线不显示的问题。
- 修复 GeoData 下载提示误报的问题。
- 修复 WebDAV 备份可能包含密码的问题。
- 修复 Mihomo 请求可能受系统代理影响的问题。

### 优化

- 优化订阅和规则管理卡片布局。
- 优化轻量模式设置入口。
- 降低界面内存占用。
- 加快连接页首次打开速度。
- 优化统计页图例显示，减少跳动。
- 优化多处页面提示和状态显示。
- 默认使用内置主题，减少冗余资源。
- 调整桌面端资源和运行目录结构。
- 优化 Windows 启动和资源加载兼容性。
