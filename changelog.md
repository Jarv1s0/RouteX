# 更新日志

## v4.1.0

> **⚠️ 重点提示：GeoData 数据库文件不再在安装包内预置**
>
> - 如果您的配置依赖了 GEO 分流规则，应用在启动内核时会拦截并抛出精准提示，不会引发闪退。
> - 当遇到此提示时，您只需前往**「规则 - GeoData」**页面，按需一键下载相应数据库文件即可正常使用。

### 修复

- 修复“订阅管理”页面顶部工具栏在部分场景下换行、错位的问题。
- 修复“订阅管理”和“规则”页面在部分自定义主题下显示不一致的问题。
- 修复删除正在使用的订阅后，运行配置未及时刷新的问题。
- 修复本地规则引用不存在的代理组时仍可能生效的问题。
- 修复更新功能相关依赖的安全风险。

### 优化

- DNS 设置页新增更多配置项，并保留运行时 Fallback DNS 配置。
- 本地规则页会提示不存在的代理组，运行时会自动跳过无效规则。
- 优化系统资源监控、剪贴板、文件选择、外部链接和本地目录打开体验，提升跨平台稳定性。
- 优化内核资源下载与解压流程，减少对系统命令的依赖。
- 优化状态刷新、节点测速和网络诊断性能，降低高频操作耗时。
- 优化大量代理节点订阅的解析性能，缓解卡顿和内存占用激增。
- 精简安装包体积，优化界面过渡动画，使整体交互更轻量流畅。
- 优化 WebDAV 和 GeoData 页面布局与反馈体验，界面更紧凑清晰。
- 统一日志、订阅和规则等页面顶部工具栏样式，减少自定义主题下的视觉差异。
- 优化连接列表节点标签显示效果，提升浅色和深色主题下的可读性。
- 优化连接页面交互体验与稳定性。
- 优化统计页和网络控制逻辑，提升页面响应稳定性。
- 提升订阅管理、实时状态同步和内核进程管理的稳定性。
- 优化异常提示，环境或配置异常时更容易定位问题。
