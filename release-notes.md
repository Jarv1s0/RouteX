# 更新日志

## v3.0.0

### ⚡ 优化
- 升级 `meta-json-schema` 到 `1.19.22`，补齐 Mihomo `xhttp`、`grpc-opts` 与 `trusttunnel` 相关配置的编辑器校验与补全支持。

### ✨ 新功能
- 首页侧边栏 UI 重构，并支持折叠模式。
- 订阅管理页合并覆写管理与代理集合视图。
- 常规设置新增 DNS 与域名嗅探管理弹窗。
- 默认主题调整为 CoolApk，并同步更新应用图标资源。

### ⚡ 优化
- 连接页面 UI 重构，连接卡片仅保留实时网速显示。
- 连接列表、规则列表、规则集合、日志页与网络拓扑节点/缩放控件改为复用统一主题色容器样式，深浅主题切换时背景与控件外观可自动适配。
- 首页侧边栏布局与导航继续收敛：统一卡片结构、间距、字重和选中态，精简入口文案，并补充路由规则统计与刷新入口。
- 侧边栏宽度改为固定尺寸，取消手动拖拽调宽。
- Sub-Store 改为独立侧栏卡片展示。
- 网络拓扑页按容器尺寸计算初始视口，结构不变时不重复重排。
- 主界面过渡、日志监听、代理组监听和编辑器加载链路进一步收敛，减少前台与后台开销。
- 连接页数据更新链路进一步收敛，降低连接驱动刷新带来的前台开销。
- DNS 与域名嗅探页面改为复用统一编辑器与表单逻辑。
- Windows 提权、自启、资源准备和 Linux 打包链路统一到 `routex` / `routex-service` / `routex-run` 命名。
- Mihomo 更新、TrafficMonitor 下载和资源准备脚本统一改为按 release 资产自动匹配，`pnpm build` / `pnpm verify` 会自动准备运行时资源。
- 统计页隐藏服务提供商用量面板，并避免首屏加载被配置读取阻塞。

### 🛠 修复
- 修复悬浮窗显隐后托盘菜单状态不同步的问题。
- 修复从首页侧栏触发规则集合刷新后，规则集合页列表不更新的问题。
- 修复节点名称国旗匹配错误，以及部分国旗字符导致的名称解析异常。
- 修复切换到网络拓扑页时的尺寸回弹与短暂白屏问题。
- 修复独立覆写入口重复的问题，旧 `/override` 路由改为跳转到订阅管理页。
- 修复 `electron-builder` 传递依赖安全告警，并修复 Windows 下资源准备遇到文件占用时直接中断的问题。
- 修复定时刷新配置时覆盖最新设置的问题，并修正侧边栏卡片状态迁移后的显示异常。
- 升级 `vite` 到 `7.3.2`，补齐最新一轮 Dependabot 安全告警修复。
